The following sample application shows how to use the HTTP Mocking Framework API to create a HTML display of a REST API documentation.

Angular Toolbox provides the AtxMockDocumentation component to render documentation of an HttpMockConfig object:

<atx-mock-documentation [config]="myConfig"/>
