Dialog API

Experimental

The Dialog API allows to create modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow.
This component is part of the LAF-less API.

Principle

Regarding the LAF-less API, dialog containers allow to create modal and non-modal popup windows by separating the popup UI logic from the app business logic.

You typically use the Dialog API to create both, blocking and non-blocking messages (such as alerts), contextual input components and so on.
