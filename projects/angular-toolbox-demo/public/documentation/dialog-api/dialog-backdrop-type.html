Dialog API

Since v 1.4.6 Experimental

The Dialog API allows to create modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow.
This component is part of the LAF-less API.

DialogBackdropType Enum

The DialogBackdropType enum defines the possible values used to assign properties of type of DialogBackdrop.

Value Description
STATIC Indicates that the dialog element is of type of "static".
MODAL Indicates that the dialog element is of type of "modal".
