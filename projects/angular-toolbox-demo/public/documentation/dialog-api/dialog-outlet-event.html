Dialog API

Since v 1.4.6 Experimental

The Dialog API allows to create modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow.
This component is part of the LAF-less API.

DialogOutletEvent

Event that indicates the state changes of components managed by the DialogService service singleton.

Constructor Function

constructor(state: DialogOutletEventType, config: DialogConfig | null = null)

Creates a new DialogOutletEvent instance.

Parameters
state: DialogOutletEventType

The state of the dialog element.
See the DialogOutletEventType API for details.

config: DialogConfig | null

The dialog element config associated with this event.
See the DialogConfig API for details.

DialogOutletEvent.SHOW

static readonly SHOW: DialogOutletEventType

The type of event dispatched each time the dialog element is displayed.

DialogOutletEvent.HIDE

static readonly HIDE: DialogOutletEventType

The type of event dispatched each time the dialog element is removed.

DialogOutletEvent.state

readonly state: DialogOutletEventType

A DialogOutletEventType value that indicates the state of the dialog element.
See the DialogOutletEventType API for details.

DialogOutletEvent.config

readonly config: DialogConfig | null

The dialog element config associated with an event of type of DialogOutletEvent.SHOW.
config is set to null when the event is of type of DialogOutletEvent.HIDE.
See the DialogConfig API for details.
