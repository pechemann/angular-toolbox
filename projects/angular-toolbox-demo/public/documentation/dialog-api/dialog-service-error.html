Dialog API

Since v 1.4.6 Experimental

The Dialog API allows to create modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow.
This component is part of the LAF-less API.

DialogServiceError

The DialogServiceError class represents errors thrown by the ATX framework when a dialog error occurs.

Constructor Function

constructor(message: string)

Creates a new DialogServiceError instance.

Extends the Error class. See the Error class reference for details.

Parameters
message: string

A human-readable description of the error.
