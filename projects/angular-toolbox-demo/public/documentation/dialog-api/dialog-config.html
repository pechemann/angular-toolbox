Dialog API

Experimental

The Dialog API allows to create modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow.
This component is part of the LAF-less API.

DialogConfig Interface

Defines the configuration associated with a custom component displayed through the Dialog API.

DialogConfig.backdrop

backdrop?: DialogBackdrop | DialogBackdropType

An optional string that indicates whether the dialog is modal, or static.
Possible values are constants of the DialogBackdropType enum:

  • If value is DialogBackdropType.MODAL, the modal can be closed by clicking outside the custom component.
  • If value is DialogBackdropType.STATIC, the modal cannot be closed by clicking outside the custom component.

If omitted, the framework uses the DialogBackdropType.MODAL constant as default value.
