Dialog API

Experimental

The Dialog API allows to create modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow.
This component is part of the LAF-less API.

DialogBackdrop Type

The DialogBackdrop type defines the possible values for the backdrop property of the DialogConfig interface.

Type Definition

type DialogBackdrop =  "modal" | "static"
