ATX Navbar

Since v 1.5.3 Experimental

Navbar components are responsive navigation headers that include support for branding, navigation and mobile collapsing menu.
This component is part of the LAF-less API.

NavbarComponent

The ATX NavbarComponent component allows to create top side menus with branding information.

The content of the branding section is defined by Developer by adding the brand attribute to an HTML container element. The toggle button content, usually an burger menu icon, is identified by the icon attribute.

All navigation links should be wrapped within LI elements, since they are direct children of an UL element.

Anything not wrapped by both, the brand and the icon attributes, is displayed in the menu section:

NavbarComponent.brandLabel

@Input() brandLabel!: string

A string that represents the aria label of the brand item.
See the ARIA Documentation for details.
Default value is undefined.

NavbarComponent.expandedLabel

@Input() set expandedLabel(value: string)
         get expandedLabel(): string

A string that represents the aria label of the toggle button when the menu is expanded in responsive mode.
See the ARIA Documentation for details.
Default value is undefined.

NavbarComponent.collapsedLabel

@Input() set collapsedLabel(value: string)
         get collapsedLabel(): string

A string that represents the aria label of the toggle button when the menu is collapsed in responsive mode.
See the ARIA Documentation for details.
Default value is undefined.

NavbarComponent.breakpoint

Since v 1.5.4 
@Input() set breakpoint(value: number)
         get breakpoint(): number

Gets or sets the breakpoint that is used to indicate whether the navbar is in responsive mode, or not.
Default value is 768.

NavbarComponent.open()

open(): void

Forces the menu to open in responsive mode.

NavbarComponent.close()

close(): void

Forces the menu to close in responsive mode.

NavbarComponent.isOpen()

isOpen(): boolean

Returns a boolean that indicates whether the menu is currently in responsive mode (true), or not (false).

Returns
boolean

Returns true whether the menu is in responsive mode; false otherwise.

NavbarComponent.isResponsiveMode()

isResponsiveMode(): boolean

Returns a boolean that indicates whether the menu is currently in responsive mode (true), or not (false).

Returns
boolean

Returns true whether the menu is in responsive mode; false otherwise.

Events

NavbarComponent.stateChange

@Ouput() readonly stateChange: EventEmitter<boolean>

Emits a event each time the the component is expanded, or collapsed in responsive mode.
