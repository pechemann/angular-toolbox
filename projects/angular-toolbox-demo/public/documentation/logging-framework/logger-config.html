Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

LoggerConfig Interface

The LoggerConfig interface defines properties for the LoggerService singleton configuration.

LoggerConfig.logConnector

logConnector?: LogConnector

Defines the log connector to be used by the LoggerService singleton.
If no log connector is specified, the LoggerService singleton the logger uses the instance defined by the DEFAULT_LOG_CONNECTOR constant.
