Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

Logger Interface

The Logger interface defines the base API for of all logging services in the ATX logging framework.

Logger.minLogLevel

minLogLevel: LogLevel

Defines the minimum log lovel at which logs must be processed.
See the LogLevel enum reference for details.

Logger.getLogs()

getLogs(): Log[]

Returns the list of logs processed by this logging service.

Returns
Log[]

A list that contains all logs send to this logger.

Logger.clearLogs()

clearLogs(): void

Removes all the logs that have been processed by this logger.

Logger.info()

info(caller: string | any, message: string, metadata?: any): void

Sends an information log message to the service.

Parameters
caller: string | any

The reference to the caller object that sends the log.

message: string

The log message associated with the log to send.

metadata?: any

Optional metadata associated with the log to send.
