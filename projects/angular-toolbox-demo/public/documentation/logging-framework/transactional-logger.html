Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

TransactionalLogger Interface

Transactional loggers allow to create proxy via the LogConnector interface to send logs to third-party tools.

Logger.setLogConnector()

setLogConnector(value: LogConnector | null): void

Sets a new log connector this transactional logger.

Parameters
value: LogConnector | null

The reference to the new log connector associated with this transactional logger.
If null the logger uses the instance defined by the DEFAULT_LOG_CONNECTOR constant.

Logger.getLogConnector()

etLogConnector(): LogConnector

Returns the reference to the log connector associated with this transactional logger.

Returns
LogConnector

The log connector associated with this transactional logger.
