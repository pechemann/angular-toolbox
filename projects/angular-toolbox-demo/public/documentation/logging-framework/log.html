Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

Log Interface

The Log interface defines the base API for of all logs of the the ATX logging framework.

Log.caller

readonly caller: string

Provides a human readable reference for the invoker associated with this log.

Log.message

readonly message: string

The message associated with this log.

Log.level

readonly level: LogLevel

The criticality level of this log.
See the LogLevel enum reference for details.

Log.timestamp

readonly timestamp: number

The timestamp at which the log has been created. (The number of milliseconds elapsed since the epoch, which is defined as the midnight at the beginning of January 1, 1970, UTC.)

Log.metadata

readonly metadata?: any

Optional metada associated with this log.
Metadata are vanilla JavaScript objects, set by Developer to provide custom objects representations.
