Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

ATX_LOGGER_CONFIG Provider

The default provider for the LoggerService singleton configuration.

Usage

You typically define the custom properties in the main NgModule declaration, or the ApplicationConfig object, to initialize the app version.
