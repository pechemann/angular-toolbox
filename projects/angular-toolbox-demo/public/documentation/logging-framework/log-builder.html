Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

LogBuilder Class

A static builder that creates new Log objects.

LogBuilder.build()

static build(caller: string, message: string, level: LogLevel = LogLevel.INFO, metadata?: any): Log

Builds and returns a new Log object initialized with the specified parameters.

Parameters
caller: string

The reference to the object that sends the new log.

message: string

The message of the new log.

level: LogLevel = LogLevel.INFO

The level of the new log.
Default value is LogLevel.INFO.

metadata?: any

The optional metadata associated with the new log.

Returns
Log

A new Log object.
