Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

LogConnector Interface

The markup interface for Angular Toolbox log connectors.
ATX log connectors are controllers that apply specific processing to logs (e.g sending logs to a data base, etc.).

LogConnector.init()

init(logList: Log[]): void

Initializes the controller.

Parameters
logList: Log[]

A list of logs used to initialize this connector.

LogConnector.clearLogs()

clearLogs(): void

Removes all logs from this connector.

LogConnector.sendLog()

sendLog(log: Log): void

Sends the specified log to this connector for processing.

Parameters
log: Log

The log to process.

LogConnector.copyLogs()

copyLogs(): void

Creates a copy of all logs processed by this connector. The way the logs are copied depends on the connector implementation.
