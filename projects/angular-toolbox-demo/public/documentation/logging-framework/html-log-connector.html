Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

HtmlLogConnector Class

An implementation of the LogConnector interface that processes logs as HTML renderable elements.

constructor

constructor(target: HTMLElement, injectStyle: boolean = true)

Creates a new HtmlLogConnector instance.

Parameters
target: HTMLElement

The HTML element where the logs will be rendered.

injectStyle: boolean

Indicates whether the ATX default HTML styles are injected into the current document (true), or not (false).
Default value is true.
