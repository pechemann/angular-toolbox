Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

ATX Constant Strings

A list of constants used as core references by the ATX Logging Framework.

Constant Description
LOG_INFO_STRING A convenient string used to identify logs of level LogLevel.INFO.
LOG_CONFIG_STRING A convenient string used to identify logs of level LogLevel.CONFIG.
LOG_WARNING_STRING A convenient string used to identify logs of level LogLevel.WARNING.
LOG_ERROR_STRING A convenient string used to identify logs of level LogLevel.ERROR.
