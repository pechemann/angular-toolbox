Angular Toolbox Logging Framework

The ATX Logging Framework provides an easy-to-use and highly configurable solution to add a logging system to your Angular libraries and applications.

Since v 0.12.0

LogLevel Enum

Defines the level of an Angular Toolbox log.

Vamue Description
INFO Indicates that the log is a basic information message.
CONFIG Indicates that the log represents a config message.
WARNING Indicates that the log is a warning message.
ERROR Indicates that the log message represents an error.
OFF A special level used by loggers to prevent logs to be processed.
