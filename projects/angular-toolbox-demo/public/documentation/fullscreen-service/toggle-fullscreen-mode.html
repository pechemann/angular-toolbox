FullscreenService.toggleFullscreenMode() Method

Toogles the fullscreen mode state of the application.

Signature

toggleFullscreenMode(target: Element | null = null, options?: FullscreenOptions): Promise<void>
Parameters
target: Element | null = null

Can be either an Element object, or null.

Default value is null.

  • if null, the service makes the entire HTML document be displayed in fullscreen mode.
  • if an HTML Element, the services make only this element be displayed in fullscreen mode.
options?: FullscreenOptions

The optional settings to pass to the fullscreen mode API.
See Element:requestFullscreen()#options on the MDN website for details.

Default value is undefined.

Returns
Promise<void>

A Promise which is resolved with a value of undefined when the transition to full screen is complete.

Exceptions

Rather than throw a traditional exception, the toggleFullscreenMode() method announces error conditions by rejecting the Promise it has returned. The rejection handler receives one of the following exception values:

TypeError

The TypeError exception may be delivered in any of the following situations:

  • The document containing the element is not fully active; that is, it is not the current active document.
  • The element is not contained by a document.
  • The element is not permitted to use the fullscreen feature, either because of Permissions Policy configuration or other access control features.
  • The element and its document are the same node.
  • The element is a popover that is already being shown via HTMLElement.showPopover().
