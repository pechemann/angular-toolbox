FullscreenService.isFullscreenModeActive Property

The FullscreenService.isFullscreenModeActive property is a boolean readonly value that indicates whether the fullscreen mode is active (true), or not (false).

Signature

get isFullscreenModeActive(): boolean
Type
boolean

Indicates whether the fullscreen mode is active (true), or not (false)
