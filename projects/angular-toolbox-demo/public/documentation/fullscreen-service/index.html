Fullscreen Service

Since v 1.8.0

A lightweight service that provides easy-to-use fullscreen mode functionalities for your Angular application.

Description

The ATX FullscreenService allows to present a specific Element in fullscreen mode, and to exit fullscreen mode once it is no longer needed.

This service has built-in features that automatically bind external full-screen calls, such as pressing the F11 key, to your presentation layer.

API

Property Description Link
change The callback event that is triggered when the fullscreen mode has changed.
isFullscreenModeActive A boolean value that indicates whether the application is displayed in fullscreen mode (true), or not (false).
Method Description Link
toggleFullscreenMode() Toogles the app fullscreen mode.
