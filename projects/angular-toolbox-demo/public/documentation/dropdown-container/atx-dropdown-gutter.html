Dropdown Container

Since v 1.3.2

Dropdown containers allow Users to toggle the display of a specific content by clicking a button.

--atx-dropdown-gutter CSS Property

The --atx-dropdown-gutter CSS property allows to set space between the button and the content container of dropdown containers. Yout typically declare it at the root of your application.
