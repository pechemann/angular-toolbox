Dropdown Container

Since v 1.3.2

Dropdown containers allow Users to toggle the display of a specific content by clicking a button.

DropdownHorizontalPosition Type

Defines the content vertical horizontal of a dropdown component. Considering the button bounds rectangle, possible values are:

  • "start": the dropdown content container right position is is equal to bounds.x.
  • "left": the dropdown content container left position is is equal to bounds.x.
  • "center": the dropdown content container center position is is equal to bounds.width/2.
  • "right": the dropdown content container right position is is equal to bounds.width.
  • "end": the dropdown content container left position is is equal to bounds.width.
