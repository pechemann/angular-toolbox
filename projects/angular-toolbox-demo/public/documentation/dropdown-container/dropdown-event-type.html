Dropdown Container

Since v 1.3.2

Dropdown containers allow Users to toggle the display of a specific content by clicking a button.

DropdownEventEventType Enum

The LayoutDragEventType enum contains all values that defines the type of a DropdownEvent instance.

DropdownEventEventType.TOGGLE

TOGGLE = "toggle"

Defines an event fired on a dropdown component just after it is shown or hidden.

DropdownEventEventType.BEFORE_TOGGLE

BEFORE_TOGGLE = "beforetoggle"

Defines an event fired on a dropdown component just before it is shown or hidden.
