Dropdown Container

Since v 1.3.2

Dropdown containers allow Users to toggle the display of a specific content by clicking a button.
This component is part of the LAF-less API.

DropdownComponent Component

The ATX DropdownComponent component allows to create popup containers that are displayed by clicking on a button.

The content of the container is defined by Developer by adding the content attribute to an HTML container element. Anything not wrapped by the content attribute is displayed in the button rendering:

DropdownComponent.buttonClass

@Input() buttonClass!: string

Sets the dropdown button CSS class.

DropdownComponent.containerClass

@Input() containerClass!: string

Sets the dropdown conpoment CSS class.

DropdownComponent.vPos

@Input() vPos: DropdownVerticalPosition = "bottom"

Sets the vertical position of the content container.
Default value is "bottom".
See the DropdownVerticalPosition API for details.

DropdownComponent.hPos

@Input() hPos: DropdownHorizontalPosition = "left"

Sets the horizontal position of the content container.
Default value is "left".
See the DropdownVerticalPosition API for details.

DropdownComponent.hideContent

hideContent(): void

Hides the dropdown container.

DropdownComponent.showContent

showContent(): void

Displays the dropdown container.

DropdownComponent.isOpened

isOpened(): boolean

Returns the display state of the dropdown.

Returns
boolean

Returns true whether the dropdown is opened; false otherwise.

Events

DropdownComponent.toggle

@Ouput() readonly toggle: EventEmitter<DropdownEvent>

Emits events just before the dropdown component is shown or hidden.
See the DropdownEvent API for details.

DropdownComponent.beforeToggle

@Ouput() readonly beforeToggle: EventEmitter<DropdownEvent>

Emits events just after the dropdown component is shown or hidden.
See the DropdownEvent API for details.
