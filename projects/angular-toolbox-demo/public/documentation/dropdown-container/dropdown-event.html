Dropdown Container

Since v 1.3.2

Dropdown containers allow Users to toggle the display of a specific content by clicking a button.

DropdownEvent

An event that represents a user interaction with a Dropdown component.

Constructor Function

constructor(target: DropdownComponent, type: DropdownEventType, newState: DropdownState, oldState: DropdownState)

Creates a new DropdownEvent instance.

Parameters
target: DropdownComponent

The dropdown container that triggered this event.

The type of this event.
