Dropdown Container

Since v 1.3.2

Dropdown containers allow Users to toggle the display of a specific content by clicking a button.

PopoverState Type

Since v 1.3.3

Defines the "popover" state of the content container whithin dropdown component.

Dropdown containers that have the "auto" state can be "light dismissed" by selecting outside the popover area.
Dropdown that have the "manual" state must always be explicitly hidden.
Possible values are:

  • "auto": Indicates that the dropdown content container can be "light dismissed".
  • "manual": Indicates that the dropdown content container must always be explicitly hidden.
