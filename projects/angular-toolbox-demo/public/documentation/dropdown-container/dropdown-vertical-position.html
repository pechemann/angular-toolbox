Dropdown Container

Since v 1.3.2

Dropdown containers allow Users to toggle the display of a specific content by clicking a button.

DropdownVerticalPosition Type

Defines the content vertical position of a dropdown component. Considering the button bounds rectangle, possible values are:

  • "start": the dropdown content container top position is is equal to bounds.y.
  • "top": the dropdown content container bottom position is is equal to bounds.y.
  • "middle": the dropdown content container middle position is is equal to bounds.height/2.
  • "bottom": the dropdown content container top position is is equal to bounds.height.
  • "end": the dropdown content container bottom position is is equal to bounds.height.
