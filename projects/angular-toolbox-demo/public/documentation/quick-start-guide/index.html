Angular Toolbox Quick Start Guide

Angular Toolbox is a library of lightweight utilities gathered to solve recurring issues encountered by Angular apps developers.

The library focuses on ease-of-use and robust test processes, to maintain quality over time.

Running Tests

Angular Toolbox unit testing is the process of testing all pieces of code provided with the library.

Run the the unit testing suite with the npm test command: 

$npm test

Testing Troubles

Running test suites when the browser window is minimized can produce failures for tests involving GUI manipulation (eg. SrcollService test suite).

This is not an Angular Toolbox issue and will be fixed by running tests with the browser window maximized, or reduced.

BorderLayoutRenderer Testing Issue

The spec BorderLayout resizing a container should throw an error if the main layout container has not been defined has no expectations.

It seems that Jasmine is not able to correctly detect the error thrown after the event emitter is fired.
This is not an Angular Toolbox issue: uncommenting the expectations should throw the error.
