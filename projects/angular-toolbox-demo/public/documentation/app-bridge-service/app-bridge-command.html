AppBridgeCommand Interface

The markup interface for all JavaScript commands consumed by the AppBridge service.

Description

Developer can creates its own JavaScript methods to create unidirectional interactions from dynamically HTML documents and an Angular app.

Such a method is called a Command. The AppBridgeCommand interface simply defines the type of a JavaScript command.

Signature

type AppBridgeCommand = (...rest: any[])=> void;
Parameters
...rest: any[]

Optional liste of user-defined paramters passed to the command when it is invoked by the framework.

Usage
