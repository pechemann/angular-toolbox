AppBridge.getCommand() Method

The AppBridge.getCommand() returns a JavaScript command previously referenced with the declareCommand() method.

Signature

getCommand(name: string): AppBridgeCommand | undefined
Parameters
name: string

The name of the JavaScript command to retrieve.

Returns

The reference to the AppBridgeCommand associated with name parameter.
undefined wether no command is associated with name parameter.
See AppBridgeCommand interface.
