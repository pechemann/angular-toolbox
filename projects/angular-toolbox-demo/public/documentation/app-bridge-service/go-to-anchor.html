AppBridge.goToAnchor() Method

The AppBridge.goToAnchor() allows to to jump to a specific part of a page.

Signature

goToAnchor(event: Event): Promise<boolean>
Parameters
event: Event

The event triggered when User interact with the associated HTML element.

Returns
Promise<boolean>

A Promise that resolves to true when navigation succeeds, or false when navigation fails.
See Router.navigate API.

Exeptions
ReferenceError

Thrown if the href attribute of the associated element is not defined.

Usage
