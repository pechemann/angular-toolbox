AppBridge.deleteCommand() Method

The AppBridge.deleteCommand() allows to remove a JavaScript command previously referenced with the declareCommand() method.

Signature

deleteCommand(name: string): boolean
Parameters
name: string

The name of the JavaScript command to remove.

Returns
boolean

true whether the command existed and has been removed; false otherwise.

Exeptions
AppBridgeError

Thrown if the name parameter refers to a native method:

  • navigate
  • goToAnchor
  • declareCommand
  • deleteCommand
  • getCommand
