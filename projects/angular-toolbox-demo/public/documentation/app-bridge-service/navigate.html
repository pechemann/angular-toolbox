AppBridge.navigate() Method

The AppBridge.navigate() allows to invoke the navigate() method of the Angular app router from an HTML page dynamically injected.

Signature

navigate(commands: string[], extras?: NavigationExtras | undefined): Promise<boolean>
Parameters
commands: string[]

An array of URL fragments with which to construct the target URL.

extras?: NavigationExtras | undefined

An options object that determines how the URL should be constructed or interpreted.
See https://angular.dev/api/router/NavigationExtras for details.

Returns
Promise<boolean>

A Promise that resolves to true when navigation succeeds, or false when navigation fails.
See Router.navigate API.

Usage
