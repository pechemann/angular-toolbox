HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

AtxMockDocumentation

An easy-to-use component that displays the documentation of a HttpMockConfig object. 

<atx-mock-documentation [config]="MOCK_CONFIG"/>

AtxMockDocumentation.title

@Input() title: string

Specifies the title of the current mocked API.

AtxMockDocumentation.config

@Input() set config(value: HttpMockConfig)
         get config(): HttpMockConfig

Gets or sets the HttpMockConfig object for which to display documentation.
