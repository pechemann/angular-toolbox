HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMockEndpointDescriptor Interface

Since v 1.4.1

HttpMockEndpointDescriptor objects provide the API to create documentation for a specific endpoint.

HttpMockEndpointDescriptor.description

description?: string

The description of the endpoint associated with this descriptor. (Supports rendering of HTML tags.)
This property is optional.

HttpMockEndpointDescriptor.params

params?: HttpParameterDescriptor[]

A collection of all parameter descriptions for the endpoint associated with this descriptor.
See the HttpParameterDescriptor interface reference for details.
This property is optional.
