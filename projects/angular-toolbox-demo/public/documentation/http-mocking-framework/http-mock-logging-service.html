HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMockLoggingService Class

Since v 0.12.2

The utility service that manages the HTTP Mocking Framework logs.

HttpMockLoggingService.size

get size(): number

Returns the number of logs currently processed by this service.

HttpMockLoggingService.info()

info(metadata: HttpMockLoggingMetadata): Log

Processes information logs with the specified metadata.

Parameters

The metadata associated with the HTTP method to log.

Returns

The new info Log object created from the specified metadata.
