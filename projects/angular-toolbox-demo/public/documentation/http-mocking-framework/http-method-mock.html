HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMethodMock Interface

The HttpMethodMock interface defines the behavior of actions invoked each time the Angular HttpClient service sends requests to a specific API endpoint.

HttpMethodMock.data()

data: (request: HttpRequest<any>, parameters?: HttpMockParameters) => HttpResponseMock

This method emulates a successful HTTP call response.

Parameters
request: HttpRequest<any>

The request associated with the intercepted HTTP call.
See the HttpRequest reference for more details.

parameters?: HttpMockParameters

The list of route parameters associated with the intercepted request.
See the HttpMockParameters interface reference for details.

Returns
HttpResponseMock

A user-defined HttpResponseMock object.
See the HttpResponseMock interface reference for details.

HttpMethodMock.description

Since v 1.4.1 
description?: string

The description of the API associated with this HTTP method.
This property is optional.
