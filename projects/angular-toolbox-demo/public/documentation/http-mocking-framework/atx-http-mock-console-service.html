HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

AtxHttpMockConsoleService Class

Since v 0.12.3

Provides functionality to display the ATX monitoring console within a new window.

HttpMockLoggingService.open()

open(): AtxHttpMockConsolePopup | null

Opens the ATX monitoring console within a new window and returns reference objects to control it.

Returns
AtxHttpMockConsolePopup

An AtxHttpMockConsolePopup object, or null when the popup is already opened.
See the AtxHttpMockConsolePopup interface reference for details.

HttpMockLoggingService.close()

close(): void

Closes the ATX monitoring console currently opened within a popup window.
