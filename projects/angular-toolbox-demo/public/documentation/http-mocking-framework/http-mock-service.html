HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMockService Class

The HttpMockService class provides the API for managing HttpMockConfig objects.

HttpMockService.addConfig()

addConfig(config: HttpMockConfig): Uuid

Adds the specified HttpMockConfig object to this HttpMockService instance.

Parameters
config: HttpMockConfig

The HttpMockConfig object to add to this HttpMockService instance.

Returns
Uuid

The Uuid instance associated with the added HttpMockConfig object.

Exeptions
HttpMockServiceError

Thrown if a config has already been registered with the UUID of the specified HttpMockConfig object.

SyntaxError

Thrown if an origin property declared within the specified HttpMockConfig object ends with a slash (/) character.

HttpMockService.getAppOrigin()

getAppOrigin(): string

Returns the current app URL origin.

Returns
string

A string that represents the current app URL origin.

HttpMockService.removeConfig()

removeConfig(uuid: Uuid): void

Removes the configuration registered with the specified UUID from this HttpMockService instance.

HttpMockService.clearConfigs()

clearConfigs(): void

Removes all registred configurations from this HttpMockService instance.

HttpMockService.hasRegisteredConfig()

hasRegisteredConfig(uuid: Uuid): boolean

Returns a boolean value that indicates whether a config with the specified UUID has been registered (true), or not (false).

Parameters
uuid: Uuid

The UUID of the config to check.

Returns
boolean

true whether a config with the specified UUID has been registered; false otherwise.

HttpMockService.getRouteConfig()

getRouteConfig(url: URL, method: HTTPMethodRef): RouteMockConfig | undefined

Returns the RouteMockConfig object associated with the specified URL instance and HTTP method.

Parameters
url: URL

The URL instance for which to find a route config.

method: HTTPMethodRef

The HTTP method for which to find a route config.

Returns
RouteMockConfig | undefined

The RouteMockConfig object associated with the specified URL instance and HTTP method whether it exists; undefined otherwise.
