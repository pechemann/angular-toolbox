HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpResponseMockBuilder Class

A statefull builder for creating new HttpResponseMock instances.

HttpResponseMockBuilder.delay()

delay(timer: number = NaN): HttpResponseMockBuilder

Sets the delay before the pseudo-server sends the HTTP response.

Parameters
timer: number

The value used to set the delay before the server sends the HTTP response.
Minimum value is 0 miliseconds.
Maximum value is HTTP_MOCK_MAX_DELAY (10 seconds).

When set to NaN, the builder creates a random value between 0 and HTTP_MOCK_MAX_DELAY.

Default value is NaN.

Returns
HttpResponseMockBuilder

The reference to this HttpResponseMockBuilder instance. 

delay(min: number, max: number): HttpResponseMockBuilder
Since v 0.11.4

Sets the delay before the pseudo-server sends the HTTP response. The delay is a random number between the min and max values that represents a timestamp, in milliseconds.

Parameters
min: number

The minimum value used used to define the random delay.
Minimum value is 0 miliseconds.

max: number

The maximum value used used to define the random delay.
Maximum value is HTTP_MOCK_MAX_DELAY (10 seconds).

Returns
HttpResponseMockBuilder

The reference to this HttpResponseMockBuilder instance.

HttpResponseMockBuilder.response()

response(error: HttpMockError | null = null): HttpResponseMock

Returns a new HttpResponseMock object, built from the properties set with the this HttpResponseMockBuilder instance methods.

Parameters
error: HttpMockError | null

An optional HTTP error, used bay the pseudo-server to return an HTTP error.
When null, the server returns a successful HTTP response.
Default value is null.

Returns
HttpResponseMock

A new HttpResponseMock instance.
