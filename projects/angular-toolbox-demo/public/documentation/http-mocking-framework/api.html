HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

API

Interface Description Link
HttpMockConfig An interface that let Developer define mocking strategies for Angular HTTP calls.
HttpMockInterceptor Provides the API to define mocking strategies for a specific API.
HttpMockEndpoint Provides the API to define mocking strategies for a specific endpoint.
HttpMethodMock Defines the behavior of actions invoked each time Angular sends requests to a specific API endpoint.
HttpResponseMock Defines the config of a HTTP response associated with a mock object.
HttpMockParameters The type of parameter objects associated with a HTTP call.
HttpMockError Provides the API that must be implemented by HTTP mock errors.
FetchClient The markup interface for Observable object built from the Fetch API.
HttpMockingFrameworkConfig Defines properties for the HTTP Mocking framework configuration.
HttpMockEndpointDescriptor Provides the API to create documentation for a specific endpoint.
HttpParameterDescriptor Provides the API to create documentation for a specific HTTP parameter, or query string parameter.
HttpMockMethodDescriptor Provides the API to create documentation for a specific HTTP method.
Utility Description Link
HttpResponseMockBuilder A statefull builder for creating new HttpResponseMock objects.
httpResponseMock() A utility function used to easily create new chainable HttpResponseMockBuilder instances.
HttpHeadersMockBuilder A statefull builder for creating new HttpHeaders instances.
httpHeadersMock() A utility function used to easily create new chainable HttpHeadersMockBuilder instances.
FetchClientBuilder A static utility class for building FetchClient objects.
FetchClientResponseType Defines the possible values for the responseType property of the FetchClientBuilder.buildFetchClient() method.
HttpMockProductionPolicy Defines the possible values for the productionPolicy property of the HttpMockingFrameworkConfig interface.
Service Description Link
HttpMockService The HttpMockService class provides the API for managing HTTP mock configuration objects.
HttpMockLoggingService The singleton that manages the HTTP Mocking Framework logs.
AtxHttpMockConsoleService Provides functionality to display the ATX monitoring console within a new window.
Decorator Description Link
@HttpMock A decorator that manages HTTP mock declarations according to an Angular component lifecycle.
Provider Description Link
HTTP_MOCKING_FRAMEWORK_CONFIG The default provider for the HTTP Mocking Framework configuration.
Constant Description Link
HTTP_MOCK_MAX_DELAY A constant that represents the maximum value for a delayed HTTP response.
