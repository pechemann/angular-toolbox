HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

AtxHttpMockConsolePopup Interface

Since v 0.12.3

The AtxHttpMockConsolePopup interface provides component references for the ATX monitoring console objects created with the AtxHttpMockConsoleService singleton.

AtxHttpMockConsolePopup.popup

popup: Window

The reference to the popup window created with the AtxHttpMockConsoleService singleton.
See the Window specification for details.

AtxHttpMockConsolePopup.componentRef

componentRef: ComponentRef>AtxMonitoringConsoleComponent<

The reference to the AtxMonitoringConsoleComponent instance created with the AtxHttpMockConsoleService singleton.
