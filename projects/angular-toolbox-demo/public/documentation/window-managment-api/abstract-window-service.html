    • AbstractWindowService Class

    Since v 1.2.0

    The abstract class that must be implemented by services that provide functionality to display Angular component within a new browser window.

    Class Signature

    abstract class AbstractWindowService implements Destroyable

    AbstractWindowService.windowClose

    Since v 1.2.1

    An event triggered each time a window is closed. 

    readonly windowClose: EventEmitter<Uuid>
    Parameters
    uuid: Uuid

    The Uuid object associated with the closed window.
    See Uuid for details.

    AbstractWindowService.get()

    Returns the window reference object associated with the specified Uuid object. 

    get(uuid: Uuid): WindowRef | undefined
    Parameters
    uuid: Uuid

    The Uuid object associated with the window reference to get.
    If no window is associated with the Uuid, returns undefined.
    See Uuid for details.

    Returns
    WindowRef

    A WindowRef object, or undefined.
    See WindowRef for details.

    AbstractWindowService.getAll()

    Returns all the window reference objects created by the service.

    Window reference objects do not longer exist when the associated window is closed. 
    getAll(): WindowRef<any, any>[]
    Returns
    WindowRef<any, any>[]

    A list of WindowRef objects created by this service.
    See Uuid for details.

    Destroyable.destroy()

    Removes all WindowRef objects from this service. 

    destroy(): void
    The Destroyable.destroy() method does not closes the associated windows.
