WindowFeatureState Enum

Since v 1.2.0

The WindowFeatureState specifies values used to activate, or deactivate, a feature for a browser window created by the WindowService.open() method.

WindowFeatureState.YES

YES = "yes"

The value used to activate a feature.

WindowFeatureState.NO

NO = "no"

The value used to deactivate a feature.
