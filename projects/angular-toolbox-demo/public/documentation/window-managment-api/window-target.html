WindowTarget Enum

Since v 1.2.0

The WindowTarget specifies the name of the browsing context the resource is being loaded into.

WindowTarget.SELF

SELF = "self"

Represents the current browsing context.

WindowTarget.BLANK

BLANK = "_blank"

Usually a new tab, but users can configure browsers to open a new window instead.

WindowTarget.PARENT

PARENT = "_parent"

The parent browsing context of the current one. If no parent, behaves as WindowTarget.SELF.

WindowTarget.TOP

TOP = "_top"

The topmost browsing context.

WindowTarget.UNFENCED_TOP

UNFENCED_TOP = "_unfencedTop"

Allows embedded fenced frames to navigate the top-level frame.
