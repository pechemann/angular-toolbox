WindowRef<T, U> Interface

Since v 1.2.0

The WindowRef defines the type of objects managed by the AbstractWindowService class.

WindowRef.window

window: U

The reference to a window stored within the AbstractWindowService class.
The generic parameter <U> represents the type of the window.

WindowRef.componentRef

componentRef: ComponentRef<T>

The reference to the Angular component displayed within a window.
The generic parameter <T> represents the type of the rendered component.
