WindowInit Interface

Since v 1.2.0

The WindowInit defines the APi used to initialize the windows created by the WindowService.open() method.

WindowInit.target

target?: WindowTarget

The name of the browsing context the resource is being loaded into.
This property is optional.

WindowInit.title

title?: string

The title of the new window.
This property is optional.

WindowInit.icon

icon?: string

The icon of the new window. Can be either a HTML path, or a base 64 encoded image string.
This property is optional.

WindowInit.left

left?: number

Specifies the distance in pixels from the left side of the work area as defined by the user's operating system where the new window will be generated.
This property is optional.

WindowInit.top

top?: number

Specifies the distance in pixels from the top side of the work area as defined by the user's operating system where the new window will be generated.
This property is optional.

WindowInit.width

width?: number

Specifies the width of the content area, including scrollbars. The minimum required value is 100.
This property is optional.

WindowInit.height

height?: number

Specifies the height of the content area, including scrollbars. The minimum required value is 100.
This property is optional.

WindowInit.directories

directories?: WindowFeatureState

Specifies whether the directories panel is visible (WindowFeatureState.YES), or not (WindowFeatureState.YENOS).
This property is optional.

WindowInit.titlebar

titlebar?: WindowFeatureState

Specifies whether the title bar is visible (WindowFeatureState.YES), or not (WindowFeatureState.YENOS).
This property is optional.

WindowInit.scrollbars

scrollbars?: WindowFeatureState

Specifies whether the scrollbars are activated (WindowFeatureState.YES), or not (WindowFeatureState.YENOS).
This property is optional.

WindowInit.toolbar

toolbar?: WindowFeatureState

Specifies whether the toolbar is visible (WindowFeatureState.YES), or not (WindowFeatureState.YENOS).
This property is optional.

WindowInit.location

location?: WindowFeatureState

Specifies whether the location is visible (WindowFeatureState.YES), or not (WindowFeatureState.YENOS).
This property is optional.

WindowInit.status

status?: WindowFeatureState

Specifies whether the status panel is visible (WindowFeatureState.YES), or not (WindowFeatureState.YENOS).
This property is optional.

WindowInit.menubar

menubar?: WindowFeatureState

Specifies whether the menu bar is visible (WindowFeatureState.YES), or not (WindowFeatureState.YENOS).
This property is optional.

WindowInit.center

Since v 1.2.3 
center?: boolean

Indicates whether the winbdow is centered (true), or not (false).
If true, the top and left properties are ignored.
