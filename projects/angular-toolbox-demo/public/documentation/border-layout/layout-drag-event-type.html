Border Layout

Since v 1.3.0

A border layout lays out a container, arranging and resizing its components to fit in five regions: north, south, east, west, and center.

LayoutDragEventType Enum

The LayoutDragEventType enum contains all values that defines the type of a LayoutDragEvent instance.

LayoutDragEventType.DRAG_START

DRAG_START = "dragstart"

Indicates that the dragging process is starting.

LayoutDragEventType.DRAG_STOP

DRAG_STOP = "dragstop"

Indicates that the dragging process stopped.

LayoutDragEventType.DRAGGING

DRAGGING = "dragging"

Indicates that the user is currently dragging a container handle.
