Border Layout

Since v 1.3.0

A border layout lays out a container, arranging and resizing its components to fit in five regions: north, south, east, west, and center.
This component is part of the LAF-less API.

API

Interface Description Link
LayoutConstraints Defines the layout constraints of a BorderLayoutContainer object.
Component Description Link
BorderLayout Arranges the components to fit in the five regions: east, west, north, south, and center.
BorderLayoutContainer A container that defines the region of a BorderLayout component instance.
Utility Description Link
LayoutRegion The list of layout constraints that defines the regions of a BorderLayout container.
--atx-handle-color Sets the color of all BorderLayoutContainer resizing handles.
Event Description Link
LayoutDragEvent An event that represents a drag interaction with a BorderLayoutContainer component.
LayoutDragEventType Defines the type of a LayoutDragEvent instance.
Error Description Link
LayoutRegionError Represents an error thrown by layout components when trying to manipulate invalid regions of a container.
