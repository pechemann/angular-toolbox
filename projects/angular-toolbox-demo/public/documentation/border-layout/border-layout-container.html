Border Layout

Since v 1.3.0

A border layout lays out a container, arranging and resizing its components to fit in five regions: north, south, east, west, and center.

BorderLayoutContainer Component

A container that defines the region of a BorderLayout component instance:

BorderLayoutContainer.constraints

get constraints(): LayoutConstraints
set constraints(constraints: LayoutConstraints)

Gets or sets the constraints for this BorderLayoutContainer object.
See the LayoutConstraints API for details.

BorderLayoutContainer.getSize()

getSize(): number

Returns the size of the container, in pixels.

Returns
number

The size of the container.

Internal API

BorderLayoutContainer.setSize()

setSize(size: number | undefined): void

Sets the size of this container depending on the region specified by the constraints property.
See the LayoutRegion enum API for details.

Parameters
size: number | undefined

The new size for this container, in pixels.
If undefined, the component uses the default size.
See the LayoutConstraints API for details.
