ButtonRoleDataObject Interface

Since v 1.7.0

The ButtonRoleDataObjectf<T> interface defines the API of objects passed as parameters to the ButtonRoleDirective.enter event emitter.

ButtonRoleDataObject.data

data: <T> | null

The reference to the data defined by the atxData attribute.
Default valus is null.

ButtonRoleDataObject.event

event: MouseEvent | KeyboardEvent | PointerEvent

The reference to the original user input event that triggered the directive enter event.
