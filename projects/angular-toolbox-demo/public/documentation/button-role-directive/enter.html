ButtonRoleDirective.enter Event

The ButtonRoleDirective.enter EventEmitter object fires events when the user presses the Enter key.

Signature

readonly enter: EventEmitter<ButtonRoleDataObject>
Type
EventEmitter<ButtonRoleDataObject>

Emits events containing the value passed to the event method of the HTML template.

Breaking Changes
Generic type is replaced by the ButtonRoleDataObject type. This ensures compatibility with the component's host element engine since the HostBinding decorator is marked as deprecated.
Since v 1.7.0
