VersionUtil Class

A convenient utility class for working with Version objects.

VersionUtil.stringify()

static stringify(version: Version): string

Returns a string that represents the specified Version object.

Parameters
version: Version

The Version object for which to get a string representation.

Returns
string

A string that represents the specified Version object.

VersionUtil.equal()

static equal(v1: Version, v2: Version): boolean

Compares the given Version objects.

Parameters
v1: Version

The first object to compare.

v2: Version

The second object to compare.

Returns
boolean

true whether both Version objects are equal; false otherwise.
